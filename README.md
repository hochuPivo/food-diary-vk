# Дневник питания — инструкция по использованию

## Как открыть приложение
Перейдите по ссылке [https://vk.com/app54470359](https://vk.com/app54470359). Приложение откроется в окне ВКонтакте, авторизация произойдет автоматически.

## Экран «Профиль»
1. Введите свои параметры: вес, рост, возраст и пол
2. Система автоматически рассчитает вашу дневную норму калорий, белков, жиров и углеводов
3. Данные сохранятся и будут доступны при следующем входе

## Экран «Дневник»
1. На главном экране отображаются четыре приема пищи: завтрак, обед, ужин, перекус
2. Цветные индикаторы показывают, сколько от дневной нормы вы уже выполнили
3. Чтобы добавить продукт, нажмите «Добавить продукт» в нужном приеме пищи
4. В окне поиска найдите продукт по названию и выберите его
5. Укажите вес в граммах — продукт автоматически добавится в дневник
6. Для изменения веса нажмите на иконку редактирования
7. Для удаления нажмите на иконку корзины и подтвердите действие

## Экран «Справочник»
1. Здесь представлен список всех доступных продуктов
2. Используйте поиск для быстрого нахождения нужного продукта
3. Нажмите на карточку продукта, чтобы увидеть подробное описание и пищевую ценность

## Важно
- Каждый день дневник автоматически очищается для новых записей
- Все данные сохраняются в облаке и не теряются при закрытии приложения
